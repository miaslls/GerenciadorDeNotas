## ⚙️ Install & Setup

Clone the project

```bash
> git clone https://github.com/miaslls/GerenciadorDeNotas.git
```

Go to the project directory

```bash
> cd GerenciadorDeNotas
```

Install dependencies

```bash
> npm install
```

Run the project

```bash
> npm run build
```
